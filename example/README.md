## 使用说明

### 1. 工程独立性

每个工程为独立可编译单元，无需依赖其他工程，可单独下载/编译/运行，后续新增工程也将保持此特性。

### 2. 硬件适配

- 默认适配：STM32F103系列（72MHz）+ 128×64分辨率 IIC 接口 OLED 屏；

- 硬件修改：可根据实际硬件调整 `IIC_OLED.h/.c` 中的引脚定义与通信速率。

### 3. 编译运行

- 库文件开发环境：Keil MDK5；

- 编译前：确认工程中 `sys.h/tim.h/gpio.h` 等底层文件与实际硬件匹配；

- 运行：下载程序后，通过 `Up/Down/Enter` 按键（低电平有效）操作菜单。

## 学习建议

1. 循序渐进：建议按工程编号顺序学习，从基础框架到高级扩展，逐步理解菜单库逻辑；

2. 重点关注：每个工程的核心代码均包含详细注释，重点理解API参数、使用规则与硬件交互逻辑；

3. 扩展开发：可基于 `menu_demo_advanced` 工程，整合前序功能实现自定义业务菜单。

## 备注

- 底层u8g2库的MCU/屏幕适配（如通信引脚、屏幕驱动、分辨率）由u8g2库自身完成，与本菜单库（抽象化组件）无关；

- 本菜单库仅需调整自身的显示逻辑参数（如菜单项行高、菜单显示区域尺寸）即可适配不同屏幕的显示效果，无需修改u8g2底层代码；

- 工程中弱定义函数可直接重写，无需修改菜单库源码；

- 后续将根据菜单库新增功能，补充对应演示工程，保持工程与库功能同步。
